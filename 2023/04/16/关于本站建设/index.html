Welcome to 星至艺空 !

星至艺空

何为星至艺空，星至艺空属于个人分享型网站，为广大网络爱好者分享学术资源。

希望我们的文档对你有所帮助 ♥

建站史

我们早在去年9月份计划搭建web网站分享平台，但奈于技术的未成熟，从中停止了开发，现在我们使用轻量级hexo框架进行网站的新搭建，并且持续优化和完善，未来可期，谢谢你的支持！！

技术栈

前端: html + css + javascript
js衍生：node.js
后端：Linux、git
编写：Markdown
云存储：腾讯云cos / 七牛云图床 /…
搭建：GitHub / gitee / 服务器

上线建议使用服务器啦提升加载速度,当然想要网站顺利发布还想要学习一下域名管理

关于Hexo

Hexo基于node.js进行开发，配合着主题的使用大大削减建站成本
Hexo总体对小白还算友好，开发者只需要多多阅读官方文档和主题文档即可做到开箱即用

但是建站也并非简单事，只能说坚持吧
